# FocusZen Katkı Kılavuzu (Contributing Guide)

Yeni bir yetenek mi kodladınız? Arayüzdeki renkleri mi iyileştirdiniz? Bir hatayı (bug) mı yok ettiniz? FocusZen'i güçlendirdiğiniz için teşekkürler!

## Öncelikli Olarak Aradığımız Katkılar
*   `Electron.js` penceresini (Saydamlık ve çerçevesizlik) performans açısından daha iyi duruma getiren güncellemeler.
*   "Yerel Ses Çalar" paneli. (Masaüstü için Lo-Fi ve Binaural beat müzikler entegre edebilirsiniz.)
*   Tüm uygulamanın rengini değiştirecek (Tema) motoru.

## Geliştirme Sürecine Katılmak İçin Adımlar
1.  **Fork** Bu repoyu kendi GitHub hesabınıza Fork'layın.
2.  **Klonlayın** (Clone) `git clone https://github.com/KULLANICI_ADINIZ/focus-zen.git`
3.  **Branch** Kendi özelliğiniz için yeni bir dal açın: `git checkout -b feature/muhtesem-ozellik`
4.  **Kodlayın!**
5.  **Commit:** Değişiklikleri anlaşılır bir İngilizce veya Türkçe formatında commit'leyin. `git commit -m "EKLENDİ: Görevlere düzenleme (edit) butonu eklendi."`
6.  **Push:** Kodunuzu kendi GitHub reponuza atın. `git push origin feature/muhtesem-ozellik`
7.  **Pull Request (PR) Gönderin:** Ana repoya girerek oluşturduğunuz farklılığı bize PR olarak iletin. En kısa sürede inceleyeceğiz.
